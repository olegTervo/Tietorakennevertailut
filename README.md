# Tietorakennevertailut

## Aihe
Toteutan 4 tietorakennetta:  
 - trie
 - binomial heap
 - fibonacci heap
 - puna-musta puu  

Ja tutkin missä tilanteessa kukin on paras. Vertailen niiden aikavaatimukset ja tarkoitukset.

## Ohjelmointikieli
Java

## Työn laajuus
Arviottu työnlaajuus n. 107 tuntia.
